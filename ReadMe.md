# Folder (Directory) Banane Ka Command:

```
mkdir folder-name
```
## Example:
```
mkdir myProject
```

# File Banane Ka Command:

### 1. Empty File Banane ke liye (Linux/macOS/Git Bash):
```
touch file-name.extension
```
## Example:
```
touch index.html
```

# Folder ke andar File Banana: 
```
mkdir myFolder
cd myFolder
touch app.js
```
### Ya ek line mein:
```
mkdir myFolder && cd myFolder && touch app.js
```
